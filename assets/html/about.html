About Me

I'm a 23-year-old Indian male who just completed his master's degree. In my department, I am a Gold Medalist, something I am quite proud of. I've been a passionate web developer since the beginning of my btech. I'm currently seeking for a career that will allow me to broaden my horizons, develop new skills, and motivate others.


Now, in terms of my acedemic experience, I did research on knowledge graph-based enhanced information retrieval and recommender systems during my Mtech.  for the enhancment  I constructed an ontology using Protégé and a knowledge graph using a supercomputer and Python programming, which I named KGC19 (Knowledge Graph of Covid-19 Scholarly Articles). KGC19 is my own work, which I hope to contribute to open source and the semantic web in the near future. It was exhausting at the time, but it was well worth every second of my time because I learned a lot and improved my Python as well as my problem-solving skills. I got interested in data-related tasks using Python as a result of this procedure, therefore I enrolled in the course "2022 Python Data Analysis & Visualization Masterclass" by the instructor named Colt Steele. He has amazing skills and a way of teaching.


During my bachelor's degree, I designed a real-time gesture-based app for manipulating the mouse and hiding selected folders using dynamic gestures as a BTech final project with 3 other team mates. It was an incredible experience. I've always been interested in the Python language and Web programming, so as a side project, I attended Colt Steele's The Web Developer Bootcamp on Udemy and learnt more and more. As a result, I've always wanted to put that knowledge to use and build projects, therefore I've created my own portfolio as well as a number of minor projects, which you can see in the project section or on my github.


Apart from these, I like spending time with my friends and family, and I also love to play cricket.
